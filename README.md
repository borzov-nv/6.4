### Задание 1

**Напишите ответ в свободной форме, не больше одного абзаца текста.**

Установите Docker Compose и опишите, для чего он нужен и как может улучшить лично вашу жизнь.

```Compose позволяет настраивать и запускать несколько контейнеров одной командой с использованием конфигурационного файла. Он также автоматизирует настройку хранения и связи между контейнерами. На работе давно хочу заменить старый кактус на древней железке. Теперь попробую настроить заббикс или прометей в контейнерах.```
